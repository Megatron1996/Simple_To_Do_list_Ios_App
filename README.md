# 📱 To Do List App

A modern and responsive To Do List iOS application built with SwiftUI using MVVM architecture.  
This app allows users to manage daily tasks efficiently with a clean and user-friendly interface.

---

## 🚀 Features

- ➕ Add new tasks
- 🗑 Delete tasks
- 💾 Local data persistence (tasks are saved when the app is closed and reopened)
- 🌙 Dark & ☀️ Light mode support
- 📱 Fully responsive design for all iPhone screen sizes
- 🔄 Smooth ScrollView task list
- 🚀 Simple Loading screen
- 🧱 MVVM Architecture implementation

---

## 🛠 Tech Stack

- Swift
- SwiftUI
- MVVM Architecture
- UserDefaults (Local Persistence)
- Responsive UI Design

---

## 🧠 What I Learned

- Implementing MVVM architecture in real project
- Managing state in SwiftUI
- Data persistence using local storage
- Supporting Dark & Light mode
- Creating adaptive layouts for different devices
